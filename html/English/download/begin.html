Run your first demo
"player.exe" is a part of ToyBox.
It's a useful portable program.
You can preview most of ToyBox's functions by it.

Download player beta version
Download player v0.1.0

Run "player.exe" should be create a empty window on your screen.
(Unix-like OS user run player.sh)

The content of the window depend on what script choosed by "default-config.lua".
Modify "examples/Window/main.lua" then reboot player.
You will notice something different.
For example, change the title of window.
